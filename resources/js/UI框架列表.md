## 切换框架

### 安装时选择

```bash
# Element Plus
npm run dev:element-plus

# Ant Design Vue
npm run dev:ant-design

# Naive UI
npm run dev:naive-ui

# Bootstrap
npm run dev:bootstrap

# Vali Admin
npm run dev:vali-admin
```

### 运行时切换

```typescript
import { uiRegistry } from '@multi-tenant-saas/ui-core'

// 切换到 Element Plus
uiRegistry.setActive('element-plus')
localStorage.setItem('multi-tenant-saas-ui-framework', 'element-plus')
window.location.reload()
```

### 使用 UI 框架选择器

```vue
<template>
  <UIFrameworkSelector v-model:visible="showSelector" />
</template>

<script setup>
import { ref } from 'vue'
import { UIFrameworkSelector } from '@multi-tenant-saas/ui-core'

const showSelector = ref(false)
</script>
```
